# Security Policy

## Reporting a vulnerability

Please email **security@agent-readiness.dev** with a description of the
issue, reproduction steps, and your contact info. We aim to acknowledge
within 3 business days.

Do **not** open a public GitHub issue for security-relevant bugs.

## Supported versions

We support the latest tagged release on `main`. Older releases receive
fixes only when the underlying issue is critical and the fix is small.

## Coordinated disclosure

We will work with you on a timeline that lets a fix ship before public
disclosure. Typical window: 30 days from acknowledgement.
